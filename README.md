## About The Project

![Windows Toolbox Screen Shot](https://i.imgur.com/SGeioqF.png)

Windows Toolbox is a powerful but simple, easy to use toolbox for Windows 10 / 11. 


### Built With

* [Powershell](https://github.com/PowerShell/PowerShell)
* [Powershell Pro Tools](https://ironmansoftware.com/powershell-pro-tools)



<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* Powershell 7

### Installation

1. Download `WindowsToolbox.zip` From [Releases](https://github.com/WinTweakers/WindowsToolbox/releases)
2. Extract `WindowsToolbox.zip` To A New Folder.
3. Open `WindowsToolbox.exe`
